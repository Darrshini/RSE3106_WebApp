Click “Start camera”, allow access, then point at a pedestrian crossing / light.
crossing_seg.onnx — the project's one perception model, run on the Node server via /api/infer: cyan dotted-line segmentation masks · the corridor arrow toward the vanishing point · the pedestrian-light box coloured by state ( green · clearance · red).
js/crossing.js captures webcam frames, POSTs them, draws the overlay, and runs the crossing decision FSM. The browser runs no neural net of its own.
